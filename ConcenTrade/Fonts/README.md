# Installation des polices

Pour que l'application fonctionne correctement, veuillez télécharger les fichiers de police suivants depuis Google Fonts et les placer dans ce dossier :

1. Poppins-Light.ttf
2. Poppins-Regular.ttf
3. Poppins-Medium.ttf
4. Poppins-SemiBold.ttf

Vous pouvez les télécharger depuis : https://fonts.google.com/specimen/Poppins

Assurez-vous que les fichiers sont nommés exactement comme indiqué ci-dessus.
